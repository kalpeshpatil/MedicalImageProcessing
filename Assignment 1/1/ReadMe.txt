Regarding Images. They can be found "images" folder with following file name format.

1D Plot have following file name format.
"1D_Value(Angle)deg delta_s_value(stepsize)"

Radon Transform output have following file name format.
"RaedonTf_value(stepsize)"

In all of the above cases value(:) output numerical value in string format.
